AzAd Smart Downloader is committed to protecting your privacy. This policy explains how we handle your information and what data we collect (or don't collect). The short answer: We don't collect or store any personal data.

üìä What We Collect

‚ùå What We DO NOT Collect

  • Your name, email address, or phone number

‚úÖ What We STORE Locally (On Your Device Only)

Everything stays in your browser. Nothing is sent to external servers.

  • Scan Results: File URLs, filenames, sizes, and types detected on the current page
    • Stored in your browser's local storage
    • Cleared automatically after download
    • Never transmitted to any server
  • User Preferences: Your download settings (speed, retries, folder organization)
    • Stored locally in browser storage
    • Only on your device
    • Syncs with Chrome if you use Chrome sync (optional)
  • Session Data: Temporary cache during active scanning/downloading
    • Cleared immediately after download completes
    • Never permanently stored

üîÑ How We Use Information

Page Scanning

When you click "Scan This Page":

  1. The extension analyzes the current page's HTML locally
  2. Extracts URLs of downloadable files
  3. Attempts to fetch file sizes (only for selected downloads)
  4. Stores temporary results in your browser's memory
  5. No external requests or tracking occurs

File Downloads

When you download files:

  1. Files are downloaded directly from their original sources
  2. Chrome's native download manager handles transfers
  3. Files saved to your Downloads folder
  4. No copies stored on external servers

  • Local Processing: All operations happen in your browser, not on servers
  • No External Communication: Extension doesn't contact external services
  • Sandboxed Execution: Extension runs in Chrome's isolated sandbox environment
  • No Tracking: No analytics, no user IDs, no session identifiers
  • No Ads: No third-party advertising networks
  • Open Source: Code is publicly auditable on GitHub
  • Chrome Manifest v3: Uses latest Chrome security standards
