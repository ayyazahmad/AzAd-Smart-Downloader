üìñ How to Use

1. Install & Pin

Install AzAd Smart Downloader from Chrome Web Store. Click the puzzle icon in Chrome and pin ASD to your toolbar for easy access.

2. Configure Chrome

For smooth bulk downloads, go to Chrome Settings ‚Üí Downloads and uncheck "Ask where to save each file before downloading".

3. Open & Scan

Navigate to any webpage with files you want. Click the ASD icon in your toolbar, then click Scan This Page in the popup.

4. Review Results

A new tab opens with all detected files. Preview images, filter by type (images, docs, videos, archives), and see file sizes.

5. Select & Download

Check the files you want, or use "Select All". Click Download Selected and files save to your Downloads folder, organized by source.
