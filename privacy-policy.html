AzAd Smart Downloader - Privacy Policy

Last Updated: February 4, 2026

1. Overview

AzAd Smart Downloader ("Extension") is committed to protecting user privacy. This policy explains how the Extension collects, uses, and protects information.

2. Information We Collect

The Extension does NOT collect personal information such as:

  • Name, email address, or phone number
  • Location or browsing history
  • Passwords or payment information
  • Identification of visited websites (beyond scan execution)

The Extension DOES collect and store locally on your device:

  • Scan Results: File URLs, filenames, file types, and file sizes detected on scanned pages (stored in browser local storage, NOT transmitted)
  • User Preferences: Download settings (concurrency level, retry count, ZIP bundling preference) stored locally in browser storage
  • Session Data: Temporarily cached during active scanning/downloading, cleared after completion

3. Data Storage & Retention

  • Location: All data stored locally on your device using chrome.storage.local
  • Transmission: No data is transmitted to external servers (except direct file downloads you initiate)
  • Retention: Scan results stored until next scan. User preferences persist until manually changed.
  • Retention (Downloads): Downloaded files stored in your local Downloads folder per Chrome settings
  • Automatic Cleanup: Scan data cleared after download queue completes (after 5 seconds)

4. Data Processing Activities

4.1 Page Scanning

When you click "Scan This Page":

  • Extension auto-scrolls the page to load lazy-loaded content (full page traversal)
  • Analyzes HTML elements (anchors, images, media tags, inline styles) locally
  • Extracts URLs matching file type patterns (images, documents, archives, audio, video)
  • Attempts to fetch file sizes using HTTP HEAD requests (3-second timeout per request)
  • Stores results only in your browser's local storage
  • No tracking or external communication occurs

4.2 Downloading

When you initiate downloads:

  • Files are downloaded directly from source URLs (as you selected)
  • Files organized in folder structure: hostname/filetype/filename
  • Chrome's native download manager handles the transfer
  • No data stored on external servers

5. Third-Party Services

  • External Links: Extension footer includes branded links to Facebook, Instagram, LinkedIn, X, YouTube, and AzAd Solutions website. These are user-initiated navigation (opening in new tabs).
  • File Hosts: Downloaded files come directly from their original hosting sources. The Extension does not proxy or modify requests.
  • Chrome APIs: Extension uses standard Chrome APIs: tabs, downloads, storage.local, scripting (as per manifest permissions)

6. User Control & Data Access

You have full control over Extension data:

  • Clear Data: Go to Chrome Extensions Settings ‚Üí AzAd Smart Downloader ‚Üí "Remove extension" or manually clear via Chrome Settings ‚Üí Privacy and security ‚Üí Clear browsing data (select "Storage" and "Cookies")
  • Review Stored Data: Open Chrome DevTools (F12) ‚Üí Application ‚Üí Local Storage ‚Üí chrome-extension://[extension-id]
  • Download Pause: Close the Extension panel or disable the Extension in Chrome settings

7. Security

  • Extension operates in isolated Chrome sandbox environment
  • No external servers or cloud storage used
  • Content scripts validated and sandboxed
  • Each file download respects Chrome's security warnings and HTTPS validation
  • No privilege escalation or cross-site request forgery attacks possible

8. Manifest v3 Compliance

Extension uses Chrome Manifest v3 (latest security standard):

  • Service Worker-based background execution (not persistent)
  • Restricted content script injection (limited to activeTab & scripting permissions)
  • No remote code execution or dynamic script injection above Chrome base policies

9. Permissions Explanation

Permission Why Needed
activeTab Access the currently active tab to inject scanner script
scripting Execute scan and auto-scroll scripts within page context
downloads Initiate file downloads to your Downloads folder
storage Store scan results and user preferences locally

We may update this privacy policy periodically. Changes will be reflected with an updated "Last Updated" date above. Continued use of the Extension constitutes acceptance of changes.
