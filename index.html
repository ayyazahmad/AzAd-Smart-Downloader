üìñ How to Use

Step 1: Install

Add the extension from Chrome Web Store. It will appear in your toolbar.

Step 2: Scan

Open any webpage. Click the AzAd icon and hit "Scan This Page" to find all downloadable files.

Step 3: Preview

Preview images as thumbnails, see file sizes, and filter by type (images, documents, etc).

Step 4: Select & Download

Check the files you want, click "Download Selected", and files go to your Downloads folder.
